control.phaseplot.boxgrid¶

control.phaseplot.boxgrid(xvals, yvals)[source]¶

Generate list of points along the edge of box.

points = boxgrid(xvals, yvals) generates a list of points that corresponds to a grid given by the cross product of the x and y values.

Parameters

  • xvals (1D array-like) – Array of points defining the points on the lower and left edges of the box.

  • yvals (1D array-like) – Array of points defining the points on the lower and left edges of the box.

Returns

grid – Array with shape (p, 2) defining the points along the edges of the box, where p is the number of points around the edge.

Return type

2D array
