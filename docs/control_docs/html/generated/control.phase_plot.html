(legacy) Phase plot for 2D dynamical systems.

Deprecated since version 0.10.1: This function is deprecated; use phase_plane_plot() instead.

Produces a vector field or stream line plot for a planar system. This function has been replaced by the phase_plane_map() and phase_plane_plot() functions.
