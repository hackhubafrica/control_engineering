control.combine_time_responses¶

control.combine_time_responses(response_list, trace_labels=None, title=None)[source]¶

Combine multiple individual time responses into a multi-trace response.

This function combines multiple instances of TimeResponseData into a multi-trace TimeResponseData object.

Parameters

  • response_list (list of TimeResponseData objects) – Reponses to be combined.

  • trace_labels (list of str, optional) – List of labels for each trace. If not specified, trace names are taken from the input data or set to None.

  • title (str, optional) – Set the title to use when plotting. Defaults to plot type and system name(s).

Returns

data – Multi-trace input/output data.

Return type

TimeResponseData
